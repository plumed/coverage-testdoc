      25             : //+PLUMEDOC EXPANSION_CV ECV_MULTITHERMAL_MULTIBARIC
      26             : /*
      27             : Expand a simulation to sample multiple temperatures and pressures.
      28             : 
      29             : The potential \ref ENERGY, \f$E\f$, and the \ref VOLUME, \f$V\f$, of the system should be used as ARG.
      30             : \f[
      31             :   \Delta u_{\beta',p'}=(\beta-\beta') E + (\beta p -\beta' p') V\, .
      32             : \f]
      33             : 
      34             : If instead you wish to sample multiple temperatures and a single pressure, you should use \ref ECV_MULTITHERMAL with as ARG the internal energy \f$U=E+pV\f$.
      35             : 
      36             : The STEPS_TEMP and STEPS_PRESSURE are automatically guessed from the initial unbiased steps (see OBSERVATION_STEPS in \ref OPES_EXPANDED), unless explicitly set.
      37             : The temperatures are chosen with geometric distribution (uniform in beta), while the pressures are uniformely spaced.
      38             : For more detailed control you can use SET_ALL_TEMPS and SET_ALL_PRESSURES.
      39             : The temperatures and pressures are then combined in a 2D grid.
      40             : 
      41             : You can use CUT_CORNER to avoid a high-temperature/low-pressure region.
      42             : This can be useful e.g. to increase the temperature for greater ergodicity, while avoiding water to vaporize, as in Ref.\cite Invernizzi2020unified.
      43             : 
      44             : You can reweight the resulting simulation at any temperature and pressure in chosen target, using e.g. \ref REWEIGHT_TEMP_PRESS.
      45             : A similar target distribution can be sampled using \ref TD_MULTITHERMAL_MULTIBARIC.
      46             : 
      47             : \par Examples
      48             : 
      49             : \plumedfile
      50             : ene: ENERGY
      51             : vol: VOLUME
      52             : ecv: ECV_MULTITHERMAL_MULTIBARIC ...
      53             :   ARG=ene,vol
      54             :   TEMP=500
      55             :   MIN_TEMP=270
      56             :   MAX_TEMP=800
      57             :   PRESSURE=0.06022140857*2000 #2 kbar
      58             :   MIN_PRESSURE=0.06022140857  #1 bar
      59             :   MAX_PRESSURE=0.06022140857*4000 #4 kbar
      60             :   CUT_CORNER=500,0.06022140857,800,0.06022140857*1000
      61             : ...
      62             : opes: OPES_EXPANDED ARG=ecv.* FILE=DeltaF.data PACE=500 WALKERS_MPI
      63             : \endplumedfile
      64             : 
      65             : Notice that \f$p=0.06022140857\f$ corresponds to 1 bar only when using the default PLUMED units.
      66             : If you modify them via the \ref UNITS command, then the pressure has to be rescaled accordingly.
      67             : 
      68             : */
