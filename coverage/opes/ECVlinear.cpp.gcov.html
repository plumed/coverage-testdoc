      25             : Linear expansion, according to a parameter lambda.

      27             : This can be used e.g. for thermodynamic integration, or for multibaric simulations, in which case lambda=pressure.

      29             : 
      30             : The difference in Hamiltonian \f$\Delta U\f$ is expected as ARG.
      31             : \f[
      32             :   \Delta u_\lambda=\beta \lambda \Delta U\, .
      33             : \f]
      34             : Use the DIMENSIONLESS flag to avoid multiplying for the inverse temperature \f$\beta\f$.
      35             : 
      36             : \par Examples
      37             : 
      38             : Typical multibaric simulation:
      39             : 
      40             : \plumedfile
      41             : vol: VOLUME
      42             : ecv: ECV_LINEAR ...
      43             :   ARG=vol
      44             :   TEMP=300
      45             :   LAMBDA=0.06022140857*2000 #2 kbar
      46             :   MIN_LAMBDA=0.06022140857  #1 bar
      47             :   MAX_LAMBDA=0.06022140857*4000 #4 kbar
      48             : ...
      49             : opes: OPES_EXPANDED ARG=ecv.vol PACE=500
      50             : \endplumedfile
      51             : 
      52             : Typical thermodynamic integration:
      53             : 
      54             : \plumedfile
      55             : DeltaU: EXTRACV NAME=energy_difference
      56             : ecv: ECV_LINEAR ARG=DeltaU TEMP=300

      58             : \endplumedfile
      59             : 
      60             : Notice that by defauly LAMBDA=0, MIN_LAMBDA=0 and MAX_LAMBDA=1, which is the typical case for thermodynamic integration.
